================================================================================
TROUBLESHOOTING
================================================================================

Error: "LLM server not running"
→ Start it: bash start_llm_server.sh

Error: "ModuleNotFoundError: No module named 'simple_parsing'"
→ Run: bash setup.sh

Error: "Docker daemon not accessible"
→ Start Docker Desktop or: sudo systemctl start docker

Error: Docker SDK proxy issues in WSL
→ Run: unset http_proxy https_proxy HTTP_PROXY HTTPS_PROXY


================================================================================
SUCCESS CRITERIA
================================================================================

✓ Demo agent completes without errors
✓ CTF agent completes without errors  
✓ CTF agent shows "Thought:" in output (proves it's working differently)
✓ Both produce valid trajectories
